SSDs

SSDs or Solid State Drives serve the same purpose as HDD, to store information. The only difference is that it doesn't have moving bits thus the name "Solid State". SSDs are typically used for boot drives since it delivers faster read and write than HDDs. SSD cost more than HDD so it is not ideal to use SSD for file storage.
There are 3 common types of SSDs: 2.5", M.2 SATA, M.2 NVMe
