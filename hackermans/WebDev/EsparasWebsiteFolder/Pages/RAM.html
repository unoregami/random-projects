RAMs

RAM is a data storage device that can provide fast read and write access. It is a volatile memory, meaning it loses all the stored data when power is lost. RAM contributes to the overall speed of a computer system. It has clock speed just like a processor and can also be overclocked.
The most common types of RAM are: DDR3, DDR4, DDR5
